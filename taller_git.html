Taller git 1. Instalar git bash (en windows) 2. Interactuar con la consola de windows o linux (comandos basicos, navegar entre carpetas, ver archivos, crear directorio o crear archivo, etc) 3. Crear cuenta de github 4. Configurar global username y email en el pc local 5. crear un primer repositorio 6. subir un archivo al repo a la rama master 7. Hacer primer commit y push 8. crear una rama nueva 9. adicionar nuevos archivos al repo en la nueva rama 10. hacer push a la nueva rama 11. hacer merge con la rama master
